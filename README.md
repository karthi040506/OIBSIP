# OIBSIP - Oasis Infobyte Internship

## Task 1: Iris Flower Classification

### Objective

Build a Machine Learning model to classify Iris flowers into Setosa, Versicolor, and Virginica species based on flower measurements.

### Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
* Google Colab

### Steps Performed

1. Data Loading
2. Data Exploration
3. Data Visualization (EDA)
4. Feature Importance Analysis
5. Model Training using Random Forest Classifier
6. Model Evaluation
7. Classification Report Generation
8. Confusion Matrix Analysis

### Results

* Achieved 100% accuracy on the test dataset.
* Petal length and petal width were identified as the most important features for classification.
* The model successfully classified all test samples without misclassification.
