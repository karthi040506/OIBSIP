# Task 1: Iris Flower Classification

## Objective
Build a Machine Learning model to classify Iris flowers into Setosa, Versicolor, and Virginica species based on flower measurements.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Google Colab

## Results
- Achieved 100% accuracy on the test dataset.
- Petal length and petal width were identified as the most important features.
- Successfully classified all test samples without misclassification.

# Task 2: Unemployment Analysis with Python

## Objective
Analyze unemployment trends in India using Python and identify patterns through data visualization and exploratory data analysis.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## Analysis Performed
- Data Cleaning
- Missing Value Handling
- Distribution Analysis
- Top 10 Regions with Highest Unemployment
- Urban vs Rural Analysis
- Correlation Heatmap

## Results
- Identified regions with higher unemployment rates.
- Compared unemployment patterns in urban and rural areas.
- Analyzed relationships between unemployment rate, employment, and labour participation rate.
