# OIBSIP - Oasis Infobyte Internship

## Task 1: Iris Flower Classification

### Objective

Build a Machine Learning model to classify Iris flowers into Setosa, Versicolor, and Virginica species based on flower measurements.

### Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
* Google Colab

### Steps Performed

1. Data Loading
2. Data Exploration
3. Data Visualization (EDA)
4. Model Training using Random Forest Classifier
5. Model Evaluation
6. Confusion Matrix Analysis

### Result

The model achieved 100% accuracy on the test dataset.
